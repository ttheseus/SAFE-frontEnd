Camera Initial Setup

Guide

Welcome to the SAFE customization application.

This app will allow you to quickly generate the input JSON file for your SAFE program. Please enter what the application prompts into the input boxes. Once you have finished, you may click the submit button and the file will be downloaded to your downloads folder.

After that, simply drag the file into the folder labelled 'data' in your SAFE program directory ('Merge') and you are done.
